📘 C++ Conditionals Practice Set 

This repository contains a structured set of 10 C++ problems focused on conditional statements. It is designed for beginners who have completed basic syntax and want to strengthen their understanding of decision-making in C++ using if, if-else, and else-if.

🎯 Objective

The goal of this practice set is to help you master:

if statements
if-else conditions
else-if ladder
Comparison operators (==, !=, >, <, >=, <=)
Logical operators (&&, ||)
Writing simple decision-based programs
📂 Problem List
🟢 Level 1: Basic Conditions
Even or Odd
Check whether a number is even or odd.
Positive, Negative, or Zero
Determine whether a number is positive, negative, or zero.
🟢 Level 2: Simple Comparisons
Greater Number
Take two numbers and print the greater one.
Equal or Not
Check if two numbers are equal.
🟡 Level 3: Multiple Conditions
Largest of Three Numbers
Find the largest among three numbers.
Grade System
Assign grades based on marks:
90+ → A
75–89 → B
50–74 → C
Below 50 → Fail
🟡 Level 4: Practical Applications
Divisibility Check
Check if a number is divisible by both 3 and 5.
Character Type Check
Determine whether a character is:
Alphabet
Digit
Special character
🔵 Level 5: Slightly Advanced
Leap Year Check
Determine whether a given year is a leap year.
(Hint: divisible by 4, but not 100 unless also divisible by 400)
Simple Calculator
Perform operations (+, -, *, /) based on user input.

✨ Keep practicing and building your logic step by step!
